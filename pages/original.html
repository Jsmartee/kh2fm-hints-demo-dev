! This is a dev build of the hints generator. To use the stable version, go here: Hint Generator

This build is identical to the main build with features to be tested.
Please report any bugs or issues to Jsmartee#7416 on Discord (Ping or DM). Suggestions are also welcome.

Features:

  • Do not allow proofs to be self hinted.
  • Ex: If there's a proof in Space Paranoids and Report 3 hints Space Paranoids, then Report 3 cannot be in Space Paranoids.
  • Priority is still: Proofs > Drive Forms > Torn Pages > (Thunder then Magnet if Atlantica is on and has a Proof)
  • Therefore if the Proofs, Forms, and Pages (and Magic) are already taking up 13 hints, there will possibly be no hinted hints if everything is in a different world.

Planned:

  • Prevent worlds from having their own hints.
    • Allow only for 1st visits?
  • Hints for torn pages should not be in 100 Acre Wood past Pooh's House.
